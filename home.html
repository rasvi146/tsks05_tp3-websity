Introduction

The TP3 features a system that enables the user to perform positioning and tracking of a tag.

This project is not only based on the Pozyx platform but also includes Ching-Hsiang's and Susmitas' delicate hardware design, Markus' and Rasmus' futuristic sensor fusion magic and finally Pontus' jedi GUI tricks.
